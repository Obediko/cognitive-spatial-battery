DATA FOLDER - IMPORTANT NOTICE
===============================

DO NOT STORE REAL PARTICIPANT DATA IN THIS FOLDER OR REPOSITORY.

This folder is a placeholder only. It exists to create the data/ directory in the project structure.

PARTICIPANT DATA POLICY
-----------------------
- Participant data must NEVER be committed to this GitHub repository.
- Participant data must NEVER be stored on GitHub, Netlify, or any public server.
- Data is collected locally in the browser during each session.
- At the end of each session, data must be downloaded using the export buttons and stored securely according to your institution's data management plan.
- Downloaded files should be stored on encrypted, access-controlled institutional storage.

WHY THIS MATTERS
----------------
- This repository is PUBLIC. Any file committed here is visible to the entire internet.
- Participant data, even pseudonymous, must be stored securely and in compliance with GDPR (EU), the UK Data Protection Act 2018, or your local equivalent.
- Pseudonymous IDs must be stored separately from any key file that links IDs to real participants.

IF YOU ACCIDENTALLY COMMIT REAL DATA
-------------------------------------
1. Remove the file immediately using git rm and force push.
2. Contact your data protection officer.
3. Note: GitHub retains commit history. Contact GitHub support for complete removal if needed.

PLACEHOLDER FILES ONLY
-----------------------
This README file is the only file that should exist in this folder in the repository.
